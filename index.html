Uptecs Email

The email library provides a quick and easy way to both send and receive email using Java, without having to use the java mail api. Simply drop the JAR in your projects class path and you are ready to go with some of the following examples:

Fetching all mail from inbox



The following demonstrates how to fetch all mail from an inbox.




    ImapReader imap=new ImapReader();
    imap.setHostname("example.com");
    imap.setUsername("username");
    imap.setPassword("password");

    imap.connect();
    int count=imap.readMailbox("INBOX");
    System.out.println("Found "+count+" emails.");
    for(int x=1;x<=count;x++) {
    	String message=imap.downloadMessage(x);
        // do something with message
	}
    imap.logout();
