Uptecs Email

The email library provides a quick and easy way to both send and receive email using Java, without having to use the java mail api. Simply drop the JAR in your projects class path and you are ready to go with some of the following examples:

Sending email via SMTP

Testing an SMTP server

The following code demonstrates testing a connection to an SMTP server. 

    Mail m=new Mail("smtp.example.com");
    if(m.testConnection()!=0)
        System.out.println("Problem with specified mailserver: "+m.getError());

Basic sending an email

The following demonstrates the most basic usage of sending mail. 

    Mail m=new Mail("smtp.example.com");
    int result=m.sendMail("bob@example.com","jane@example.com","My subject","content2");
    if(result>0)
        System.err.println(m.getError());

Authenticated sending an email

Generally you will probably need to authenticate with your SMTP server before attempting to send an email. 

    Mail m=new Mail("smtp.example.com");
    m.enableAuthentication("jacob@example.com","password");
    int result=m.sendMail(
		"bob@example.com",
		"jane@example.com",
		"My subject",
		"content2");
    if(result>0)
        System.err.println(m.getError());

Fetching all mail from inbox



The following demonstrates how to fetch all mail from an inbox.




    ImapReader imap=new ImapReader();
    imap.setHostname("example.com");
    imap.setUsername("username");
    imap.setPassword("password");

    imap.connect();
    int count=imap.readMailbox("INBOX");
    System.out.println("Found "+count+" emails.");
    for(int x=1;x<=count;x++) {
    	String message=imap.downloadMessage(x);
        // do something with message
	}
    imap.logout();
